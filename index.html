HTML

  • The lang="eng" selects the language tag
  • The meta tags are mostly for SEO search
  • The head element contains information about the page
  • The class / id tags are used for JavaScript

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • * symbol at start of a selector in CSS will apply the value to all the class in that selector
  • sperating 2 selectors with a comma , will allow you to select 2 selectors when applying CSS

Git

  • git status - check what branch we are currently on
  • git checkout -b [new name for example feature/starter-code] - switch to a new branch
  • git add -a - all all files to branch
  • git commit -m “[insert description]”
  • git pull origin main - fetch main branch to check there’s no conflicts and it’s all up to date to check. (git pull to make sure that you have the latest version of the project on your local computer before you start to work)
  • git push origin [insert current folder to push] - pushes to GitHub branch
